# Contributing

This is a single-operator commercial project. The source is public for
transparency and reference; it is not open for unsolicited contributions.

## What this means in practice

- **Pull requests will be closed without review.** Please open an issue
  first describing the change you'd like to make and the reason. If we
  agree the work fits, we'll discuss scope and merge expectations there
  before any code is written.
- **Do not submit fixes or features speculatively.** A merged PR transfers
  copyright assignment to the project; we cannot accept that lightly
  without prior alignment.
- **Issues are welcome.** Bug reports, security concerns, and
  questions are read and responded to within ~7 days. Format:
  - Steps to reproduce (if a bug)
  - Expected vs. actual behaviour
  - Environment (browser / OS / version)
  - Logs or screenshots if relevant

## Security

Do **not** open public issues for security findings. See
[`docs/SECURITY.md`](./docs/SECURITY.md) for the disclosure process.
