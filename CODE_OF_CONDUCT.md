# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
in spirit. The short version:

## Be respectful

Be considerate. Disagree on the technical merits. Don't make it personal.

## Don't harass

Personal attacks, slurs, sexualised language, deliberate intimidation,
doxxing, or sustained disruption of discussion are not welcome here and
will result in a ban from issues and any related channels.

## Scope

This applies to issues, comments on commits / PRs, email exchanges
about the project, and any public space where someone is representing
the project.

## Enforcement

Concerns can be reported privately to stefanromanpers@gmail.com.
Reports are read within ~7 days. The maintainer's decisions on
enforcement are final.
